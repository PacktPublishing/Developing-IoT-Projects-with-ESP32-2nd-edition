# Example for `mcp960x` driver

## Datasheet

[MCP960X/L0X/RL0X](https://ww1.microchip.com/downloads/en/DeviceDoc/MCP960X-L0X-RL0X-Data-Sheet-20005426F.pdf)

## What it does

The example configures four `mcp960x` device, shows details about each device.
It reads and shows temperature from each device in a loop.
