## Connecting

| `TSL2561` | `ESP8266` | `ESP32` | `NodeMCU` |
|-----------|-----------|---------|-----------|
| `SCL`     | `GPIO5`   | `GPIO19`| `D1`      |
| `SDA`     | `GPIO4`   | `GPIO18`| `D2`      |
